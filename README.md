##&lt;neocloud-combo&gt;

`neocloud-combo` Polymer component for display a list of items.

To develop and test:

1. Clone repository
```
git clone git@github.com:NEOGRUP/neocloud-combo.git
```
2. Test it with
```
polymer serve
```
3. Links

   [Demo page](http://localhost:8000/components/neocloud-combo/demo)
  
   [Documentation page](http://localhost:8000/components/neocloud-combo/)
